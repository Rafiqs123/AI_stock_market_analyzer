# AI_stock_market_analyzer
Interaktywna aplikacja do prognozy cen akcji i kryptowalut z wykorzystaniem Machine Learning. Integruje Random Forest, analizę techniczną (SMA, EMA, MACD, RSI), pobieranie newsów finansowych, backtesting strategii oraz wizualizację danych. Interfejs webowy z WebSocket dla real-time komunikacji. Stack: Flask, scikit-learn, yfinance, TA-Lib, Redis.
